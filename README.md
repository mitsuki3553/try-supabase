## タイトル "Post Next"

### 要件
- サインアップ・サインインができる投稿サイト　→　OAuthの実装は未定
- 自身の投稿ごとに、他者から見られる(パブリックポスト)か見られない(プライベートポスト)か選べる
- 各投稿に対してコメントができる
- 各投稿に対して"いいね"ができる

### 使用サービス
- github 
- supabase 
- vercel
